# The cm Transition Problem at the Hexagonal Boundary

## 0. The punchline

We want a user to be able to slide a single slider and watch a **cm** wallpaper
pattern deform continuously through the full sequence:

> not-well-rounded centered-rectangular → hexagonal → well-rounded centered-rectangular → square

The problem: the **cm mirror direction** is forced to jump by 30° at the
hexagonal point.  There is no path through our current lattice parameter space
that avoids this jump.  A fix likely requires enlarging the set of allowed
lattices.

---

## 1. Setup: lattice normalization

Every 2D lattice is generated by two linearly independent translations.
We normalize to:

$$\mathbf{a} = (0,\,1), \qquad \mathbf{b} = (x,\,y)$$

with the constraints

$$x \ge 0, \quad 0 \le y \le \tfrac12, \quad x^2 + y^2 \ge 1.$$

The last constraint is the Minkowski-reduction condition: **b** is no shorter
than **a**.  The allowed region in the $(x,y)$ plane is a wedge-shaped domain
bounded by the unit circle from below, $y=0$ on the bottom edge, and $y=1/2$
on the top edge.

**Lattice types** within this domain:

| Region | Lattice type |
|---|---|
| $(1,\,0)$ | Square |
| $(\sqrt3/2,\,\tfrac12)$ | Hexagonal |
| $y = 0,\ x > 1$ | Rectangular |
| $x^2+y^2 = 1,\ 0 < y < \tfrac12$ | Centered rectangular ("well-rounded") |
| $y = \tfrac12,\ x > \sqrt3/2$ | Centered rectangular ("not-well-rounded") |
| Interior ($x^2+y^2 > 1,\ 0 < y < \tfrac12$) | Oblique |

The centered-rectangular lattices live on **two disjoint 1-dimensional curves**
that meet only at the hexagonal point $(\sqrt3/2,\,\tfrac12)$.

```
  y
  ½ ──────hex●━━━━━━━━━━━━━━━ not-well-rounded (y=½)
       ╱     
      ╱  oblique interior
     ╱        
  0 ─sq●━━━━━━━━━━━━━━━━━━━━━ rectangular (y=0)
      1    1.5    2    2.5   x

  (curved arc from sq to hex = well-rounded, x²+y²=1)
```

## 2. cm on a centered-rectangular lattice

A centered-rectangular lattice always has a **rectangular sublattice** with a
centering vector at the midpoint of each rectangle.

### 2a. Well-rounded case ($x^2+y^2 = 1$)

Here $|\mathbf{a}| = |\mathbf{b}| = 1$.  The rectangular sublattice has basis

$$\mathbf{a}+\mathbf{b} = (x,\,1+y), \qquad \mathbf{b}-\mathbf{a} = (x,\,y-1),$$

which are perpendicular (check: $(\mathbf{a}+\mathbf{b})\cdot(\mathbf{b}-\mathbf{a}) = |\mathbf{b}|^2 - |\mathbf{a}|^2 = 0$).
The centering vector is $\mathbf{a}$ (or equivalently $\mathbf{b}$), which
sits at the center of each rectangle.

A **cm** pattern uses a mirror along one axis of this rectangle.  The two
choices give two direction variants:

- **Variant 0:** mirror ∥ $\mathbf{a}+\mathbf{b}$, angle $= \text{atan2}(1+y,\,x)$
- **Variant 1:** mirror ∥ $\mathbf{b}-\mathbf{a}$, angle $= \text{atan2}(y-1,\,x)$

**Why the mirror preserves the lattice.**  Because $|\mathbf{a}| = |\mathbf{b}|$,
reflection across the $\mathbf{a}+\mathbf{b}$ direction swaps $\mathbf{a}$
and $\mathbf{b}$:

$$\sigma(\mathbf{a}) = \mathbf{b}, \qquad \sigma(\mathbf{b}) = \mathbf{a}.$$

Both images are in $L$, so $\sigma(L) = L$. ✓

**Key formula:** $\sigma(\mathbf{a}) = \mathbf{b}$ holds if and only if
$|\mathbf{a}| = |\mathbf{b}|$.  (Proof: the reflection
$\sigma$ across $\mathbf{a}+\mathbf{b}$ satisfies
$\sigma(\mathbf{a}) = \frac{2(\mathbf{a}\cdot(\mathbf{a}+\mathbf{b}))}{|\mathbf{a}+\mathbf{b}|^2}(\mathbf{a}+\mathbf{b}) - \mathbf{a}$,
and this equals $\mathbf{b}$ iff $2(|\mathbf{a}|^2 + \mathbf{a}\cdot\mathbf{b}) = |\mathbf{a}+\mathbf{b}|^2 = |\mathbf{a}|^2 + 2\mathbf{a}\cdot\mathbf{b} + |\mathbf{b}|^2$, i.e. $|\mathbf{a}|^2 = |\mathbf{b}|^2$.)

**Angles along the well-rounded arc:**

| Lattice point | $(x,y)$ | variant-0 angle | variant-1 angle |
|---|---|---|---|
| Square | $(1, 0)$ | $\pi/4 = 45°$ | $-\pi/4 = -45°$ |
| Mid-arc | $(0.954, 0.3)$ | $49.3°$ | $-36.3°$ |
| Hexagonal | $(\sqrt3/2,\,\tfrac12)$ | $\pi/3 = 60°$ | $-\pi/6 = -30°$ |

The mirror angle varies **continuously** from 45° to 60° along this arc. ✓

---

### 2b. Not-well-rounded case ($y = \tfrac12$, $x > \sqrt3/2$)

Here $|\mathbf{a}| = 1$ but $|\mathbf{b}| = \sqrt{x^2 + 1/4} > 1$.
The vectors $\mathbf{a}$ and $\mathbf{b}$ have **unequal length**.

The rectangular sublattice has basis

$$\mathbf{a} = (0,\,1), \qquad 2\mathbf{b} - \mathbf{a} = (2x,\,0),$$

which are perpendicular ($\mathbf{a}$ is vertical, $2\mathbf{b}-\mathbf{a}$ is horizontal).
The centering vector is $\mathbf{b} = (\mathbf{a} + (2\mathbf{b}-\mathbf{a}))/2$,
sitting at the center of each rectangle.

The correct **cm** mirrors on this lattice are along the rectangle axes:

- **Variant 0:** mirror along $\mathbf{a}$ (vertical), angle $= \pi/2 = 90°$
- **Variant 1:** mirror along $2\mathbf{b}-\mathbf{a}$ (horizontal), angle $= 0°$

**Why these mirrors preserve the lattice:**

Mirror along $\mathbf{a}$ (vertical):
$$\sigma(\mathbf{a}) = \mathbf{a} \in L \checkmark, \qquad
\sigma(\mathbf{b}) = (-x,\,\tfrac12) = \mathbf{a} - \mathbf{b} \in L \checkmark$$

Mirror along $2\mathbf{b}-\mathbf{a}$ (horizontal):
$$\sigma(\mathbf{a}) = -\mathbf{a} \in L \checkmark, \qquad
\sigma(\mathbf{b}) = (x,\,-\tfrac12) = \mathbf{b} - \mathbf{a} \in L \checkmark$$

**Why the $\mathbf{a}+\mathbf{b}$ direction does NOT work** (this is the bug):

Take $x = 1.766$, $y = 0.5$.  Then $\mathbf{a}+\mathbf{b} = (1.766,\,1.5)$,
and the mirror angle is $\text{atan2}(1.5,\,1.766) \approx 40.34°$.

Reflecting $\mathbf{a} = (0,1)$ across this mirror:

$$\sigma(\mathbf{a}) = \frac{2(\mathbf{a}\cdot\mathbf{d})}{|\mathbf{d}|^2}\mathbf{d} - \mathbf{a}
= \frac{2(1.5)}{5.369}(1.766,\,1.5) - (0,\,1) \approx (0.987,\,-0.162)$$

Is $(0.987,\,-0.162)$ in $L = \{m\mathbf{a} + n\mathbf{b}\}$?
$$1.766\,n = 0.987 \;\Rightarrow\; n \approx 0.559, \qquad
m + 0.5\,n = -0.162 \;\Rightarrow\; m \approx -0.441$$

**Neither is an integer.** $\sigma(\mathbf{a}) \notin L$.

The reflection takes the lattice outside itself.  This is because
$|\mathbf{a}| \ne |\mathbf{b}|$, so the $\mathbf{a}+\mathbf{b}$ bisector
is *not* a symmetry of the lattice.

## 3. The hex point: where three worlds collide

The hexagonal lattice $(\sqrt3/2,\,\tfrac12)$ has $|\mathbf{a}| = |\mathbf{b}| = 1$,
$60°$-rotation symmetry, and **six** mirror directions spaced 30° apart.

These six mirrors naturally fall into **three pairs**, corresponding to
three different ways of viewing the hex lattice as a centered-rectangular lattice:

| Decomposition | Rectangle basis | Centering vec | Mirror angles |
|---|---|---|---|
| **#1:** $\mathbf{a}+\mathbf{b}$, $\mathbf{b}-\mathbf{a}$ | $(\sqrt3/2,\,3/2)$ and $(\sqrt3/2,\,-1/2)$ | $\mathbf{a}$ | **60°**, **−30°** |
| **#2:** $\mathbf{a}$, $2\mathbf{b}-\mathbf{a}$ | $(0,\,1)$ and $(\sqrt3,\,0)$ | $\mathbf{b}$ | **90°**, **0°** |
| **#3:** $\mathbf{b}$, $\mathbf{a}-2\mathbf{b}$ | $(\sqrt3/2,\,1/2)$ and $(-\sqrt3/2,\,1/2)$ | $\mathbf{a}-\mathbf{b}$ | **30°**, **−60°** |

At the hex point all three decompositions are valid; any of the six mirror
directions gives a legitimate cm group.

**The critical observation:**

- The **well-rounded arc** approaches hex using decomposition **#1**.
  The cm mirror angles arrive at **60°** and **−30°**.

- The **not-well-rounded line** approaches hex using decomposition **#2**.
  The cm mirror angles arrive at **90°** and **0°**.

These are **different pairs of hex mirrors**.  They do not match.

---

## 4. The discontinuity

Consider the cm pattern with "variant 0" (the first mirror direction):

| Path | Near hex | At hex | Mirror angle |
|---|---|---|---|
| Well-rounded | atan2(1+y, x) → 60° | 60° | **60°** |
| Not-well-rounded | π/2 = 90° | 90° | **90°** |

There is a **30° jump** in the mirror direction at the hex point when
switching from one path to the other.

```
Mirror angle for cm variant 0:

  90° ━━━━━━━━━━━━━━━━━━━●  not-well-rounded
                          ┊
                     30° gap
                          ┊
  60° ━━━━━━━━━━━━━━━━━━━●  well-rounded (at hex)
   ╲                     ╱
    ╲   well-rounded    ╱
     ╲    arc          ╱
  45° ●               square
```

No matter which two directions you assign to "variant 0" and "variant 1",
you cannot make both paths agree at hex.

---

## 5. Why you can't fix it within the current domain

Could we find a path through the normalized domain $(x,y)$ that transitions
smoothly from one set of cm mirrors to the other?

**No.** Here's why:

1. **cm requires a centered-rectangular (or square/hex) lattice.**
   The oblique interior of the domain does not support cm at all — a generic
   oblique lattice has no mirror symmetry.

2. **The centered-rectangular locus consists of two disjoint curves** (the arc
   $x^2+y^2=1$ and the line $y=1/2$) meeting only at hex.

3. **On each curve, the cm mirror direction is uniquely determined** (up to the
   discrete variant choice).  On the arc, it's along $\mathbf{a}+\mathbf{b}$;
   on the line, it's along $\mathbf{a}$ or $2\mathbf{b}-\mathbf{a}$.

4. **At hex, these directions are different** (60° vs 90° for variant 0).

Therefore: within the normalized domain, there is no continuous 1-parameter
family of cm groups connecting the two paths.

---

## 6. The way out: leaving the normalized domain

To create a seamless transition, we need a family of centered-rectangular
lattices where the mirror direction rotates continuously from 90° to 60°.
Such a family exists — it just doesn't fit inside the standard normalized
domain.

### 6a. What the intermediate lattices look like

A centered-rectangular lattice with equal-length basis vectors and mirror
at angle $\theta$ can be written as:

$$\mathbf{a}_\theta = (-\sin\alpha,\,\cos\alpha), \qquad
\mathbf{b}_\theta = (\sin\alpha,\,\cos\alpha)$$

where the mirror is along $\mathbf{a}+\mathbf{b} = (0,\,2\cos\alpha)$
(vertical) — wait, that gives a fixed mirror direction.

Let me be more concrete.  We want a 1-parameter family
$\theta \in [0°, 30°]$ such that:

- At $\theta = 0°$: the lattice is hexagonal with cm mirror at 90° (decomposition #2).
- At $\theta = 30°$: the lattice is hexagonal with cm mirror at 60° (decomposition #1).
- For $0° < \theta < 30°$: the lattice is centered-rectangular with cm mirror at $90° - \theta$.

**Construction.** Take the hex lattice and rotate the entire lattice (and
mirror) by angle $-\theta$:

$$\mathbf{a}_\theta = R_{-\theta}\,\mathbf{a}, \qquad
\mathbf{b}_\theta = R_{-\theta}\,\mathbf{b}$$

where $R_{-\theta}$ is counter-clockwise rotation by $-\theta$.  At each
$\theta$ the lattice is identical to the hex lattice (just rotated), and the
cm mirror is at $90° - \theta$.

But this is trivial — we're just rotating the whole picture.  The **lattice
shape** doesn't change; it's always hexagonal.

To get a *non-trivial* interpolation (where the lattice shape actually
changes from not-well-rounded to well-rounded), we need to simultaneously
deform the lattice shape and rotate the mirror.

### 6b. A concrete interpolation

Define a path in the space of (un-normalized) lattices parametrized by $t \in [0,1]$:

At $t=0$: not-well-rounded centered-rectangular at hex boundary.
- $\mathbf{a} = (0,1)$, $\mathbf{b} = (\sqrt3/2,\, 0.5)$ — hex lattice.
- cm mirror along $\mathbf{a}$ at angle 90°.

At $t=1$: same hex lattice, different decomposition.
- $\mathbf{a} = (0,1)$, $\mathbf{b} = (\sqrt3/2,\,0.5)$ — same hex lattice.
- cm mirror along $\mathbf{a}+\mathbf{b}$ at angle 60°.

For intermediate $t$, we need a centered-rectangular lattice with:
- mirror angle at $90° - 30°t$,
- $|\mathbf{a}| = |\mathbf{b}|$ (so the $\mathbf{a}+\mathbf{b}$ bisector
  is indeed a lattice symmetry).

**One way:** Rotate the basis by $-15°t$ (half the mirror rotation, since
the mirror direction is the bisector of $\mathbf{a}$ and $\mathbf{b}$,
and if both rotate by $\alpha$, the bisector rotates by $\alpha$):

$$\mathbf{a}_t = R_{-30°t}\,\mathbf{a} = (-\sin 30°t,\;\cos 30°t),$$
$$\mathbf{b}_t = R_{-30°t}\,\mathbf{b}.$$

Mirror direction: $\mathbf{a}_t + \mathbf{b}_t = R_{-30°t}(\mathbf{a}+\mathbf{b})$,
angle $= 60° - 30°t$.

At $t=0$: mirror at 60°.  At $t=1$: mirror at 30°.

Hmm — this doesn't give 90°.  The problem is that at $t=0$ we want
decomposition #2's mirror (90°), not decomposition #1's mirror (60°).
And decomposition #2 uses the **a** direction, not **a+b**.

Let's try a different approach.  At the hex point, pick basis vectors for
a centered-rectangular lattice with mirror at angle $\phi$, where $\phi$
ranges from 90° to 60°.

A centered-rectangular lattice with $|\mathbf{a}| = |\mathbf{b}|$ and
the $\mathbf{a}+\mathbf{b}$ bisector at angle $\phi$ can be constructed as:

$$\mathbf{a} = (\cos(\phi+\psi),\;\sin(\phi+\psi)), \qquad
\mathbf{b} = (\cos(\phi-\psi),\;\sin(\phi-\psi))$$

where $\psi$ is the half-angle between $\mathbf{a}$ and $\mathbf{b}$.
For the hex lattice, $\psi = 30°$ (since $\mathbf{a}$ and $\mathbf{b}$ are
60° apart when the lattice is hexagonal... but actually, in our normalization,
$\mathbf{a}=(0,1)$ and $\mathbf{b}=(\sqrt3/2, 1/2)$, and the angle between
them is $\cos^{-1}(\mathbf{a}\cdot\mathbf{b}) = \cos^{-1}(1/2) = 60°$, so
$\psi = 30°$).

With $\psi = 30°$ fixed (hex lattice shape), varying $\phi$ from 90° to 60°:

At $\phi = 90°$: $\mathbf{a} = (\cos 120°, \sin 120°) = (-1/2, \sqrt3/2)$,
$\mathbf{b} = (\cos 60°, \sin 60°) = (1/2, \sqrt3/2)$.
Mirror along $\mathbf{a}+\mathbf{b} = (0, \sqrt3)$, angle = 90°. ✓

At $\phi = 60°$: $\mathbf{a} = (\cos 90°, \sin 90°) = (0, 1)$,
$\mathbf{b} = (\cos 30°, \sin 30°) = (\sqrt3/2, 1/2)$.
Mirror along $\mathbf{a}+\mathbf{b} = (\sqrt3/2, 3/2)$, angle = 60°. ✓

This traces a path where the **same hex lattice** (just with different basis
choice) has the mirror rotating from 90° to 60°.  The lattice shape never
changes — it's hex the whole time.

**The catch:** This path has $\mathbf{a} \ne (0,1)$ for intermediate $\phi$.
The first basis vector is *rotated away from vertical*.  This is **outside
our normalized domain**, which requires $\mathbf{a} = (0,1)$.

### 6c. Extending to include shape deformation

To make the full seamless path that includes non-hexagonal lattices, glue
three pieces together:

**Piece 1:** Not-well-rounded centered-rectangular, $y = 1/2$, $x$ decreasing
toward $\sqrt3/2$.  Mirror along $\mathbf{a}$ (vertical, 90°).  Lattice in
normalized form.

**Piece 2:** At hex, rotate the basis while staying on the hex lattice,
sweeping the mirror from 90° to 60° (as in §6b).  Lattice leaves normalized
form — $\mathbf{a}$ is no longer $(0,1)$.

**Piece 3:** Well-rounded centered-rectangular, $x^2+y^2=1$, $y$ decreasing
toward 0.  Mirror along $\mathbf{a}+\mathbf{b}$, angle decreasing from 60°
to 45° (square).  Lattice in normalized form.

```
mirror angle:

  90° ━━━━━━━━━━━━━━━●         piece 1 (not-well-rounded)
                      ╲
                       ╲       piece 2 (re-index at hex, outside norm. domain)
                        ╲
  60° ━━━━━━━━━━━━━━━━━━━●     piece 3 (well-rounded, continues to square)
   ╲                     ╱
    ╲                   ╱
     ╲                 ╱
  45° ●               square
```

**Piece 2 is the problem.**  It requires either:

(a) **Allowing rotated lattices** (dropping the $\mathbf{a}=(0,1)$ normalization),
or

(b) **Re-normalizing at hex** and accepting a discontinuous change of basis
(which is mathematically fine but may look jarring in the UI), or

(c) **Designing a path through genuinely different lattice shapes** (not just
rotating the hex lattice) that interpolates between decomposition #2 and #1.

## 7. What the current code does (and why it's wrong)

The most recent commit changed `getAllowedIsometries()` to always use the
$\mathbf{a}+\mathbf{b}$ and $\mathbf{b}-\mathbf{a}$ directions for centered-rectangular lattices, regardless of whether the lattice is well-rounded.

This achieves **angle continuity** at the hex boundary: as $x \to \sqrt3/2$
from above (not-well-rounded), $\text{atan2}(1.5,\,x) \to 60°$, matching the
hex value.

But it produces **mathematically invalid cm groups** on not-well-rounded lattices.
For $x = 1.766$, $y = 0.5$:

- The mirror at 40.34° ($\mathbf{a}+\mathbf{b}$ direction) does not preserve the lattice.
- Reflecting $\mathbf{a} = (0,1)$ gives $\approx (0.987,\,-0.162)$, which requires
  $n \approx 0.559$ and is not a lattice vector.
- `generateGroup` correctly flags this with a warning:
  *"The canonical translation group of the symmetry group is not equal to the lattice L."*

The old code (before the recent commit) used vertical/horizontal for
not-well-rounded, which was **mathematically correct** but **discontinuous** at hex.

**Neither option is satisfactory.** The correct approach requires addressing
the structural issue described in §6.

## 8. Summary for the grad student

1. **cm** requires a centered-rectangular lattice. The mirror must be along
   an axis of the underlying rectangle.

2. In our normalized parameter space, centered-rectangular lattices form
   **two disjoint curves** meeting at the hex point.

3. Each curve uses a **different rectangular sub-structure**, so the cm
   mirror directions are different (90° vs 60° for variant 0).

4. At hex, both sub-structures are valid, but they correspond to **different
   mirror directions** among the six hexagonal mirrors.

5. There is **no path within the normalized domain** that continuously
   interpolates between the two mirror directions, because the oblique
   interior doesn't support cm.

6. A seamless transition requires **leaving the normalized domain** — either
   by allowing rotated lattices ($\mathbf{a} \ne (0,1)$) or by introducing
   a new parameterization that treats the hex point as a smooth junction
   rather than a corner.

7. Concretely, the transition at hex involves rotating the basis vectors of
   the hex lattice by 30° while keeping the lattice fixed, which re-indexes
   which pair of lattice vectors are "the basis" and which rectangle
   sub-structure is "the one for cm."

**The hex point is where three centered-rectangular decompositions meet.**
Our two UI paths each approach hex via a different decomposition, and
connecting them requires passing through a third decomposition that our
current normalization doesn't represent.
